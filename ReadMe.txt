Моргунова Ольга Максимовна 321 группа
OS - Ubuntu 16.04 LTS 64bit, установленная на виртуальной машине VirtualBox
Memory - 990 MiB
Processor - Intel Core i7-7500 CPU @ 2.70 GHz
Graphics - Gallium 0.4 on LLvmpipe (LLVM 4.0 256 bits)

Файл настроек:
	1 строка - 3 координаты положения источника, мощность источника и его максимальный радиус
	2 строка - 3 координаты положения камеры, 3 координаты вектора направления камеры,
			   3 координаты вектора "вверх", 3 координаты вектора "вправо",
			   вертикальный угол обзора и горизонтальный угол обзора в радианах
	3 строка - 2 числа - разрешение выходной картинки

Файлы настроек и модели лежат в папке img.

Из бонусной части реализовано распараллеливание программы средствами OpenMP.
